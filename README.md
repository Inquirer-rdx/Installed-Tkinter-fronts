A simple Python GUI application built with Tkinter that displays all installed fonts on your system 
in a scrollable list. Each font family is rendered using its own typeface for easy previewing.

Features:
---------
Lists all available font families (sorted alphabetically)
Renders each font in its native style
Smooth scrolling with mousewheel support
Clean, minimal interface


Requirements: Python 3.x with tkinter (usually included by default)
